Plugin pro zobrazení nejnovějších novinek a informací z neomezeného hostingu WEDOS, přehledně na nástěnce v administraci.

== Description ==

S tímto pluginem budete mít vždy přehled o všech novinkách a akcích z neomezeného hostingu WEDOS. Jako první se tak můžete dozvědět například o nových soutěžích, slevách nebo slevových kupónech.

= Co s tímto pluginem získáte? =

* Novinky a informace z oficiálního RSS kanálu společnosti [WEDOS.cz](http://hosting.wedos.com)
* Novinky a informace z oficiální stránky na [Google+](https://plus.google.com/u/0/103221313212683920526/posts).
* Graf aktuálního datového vytížení
* Rychlé odkazy na online zákaznický chat, nápovědu a fórum hostingu WEDOS
* Rychlé odkazy pro nákup nové domény, hostingu nebo FTP disku
* Plugin neobsahuje žádné affiliate odkazy ani žádnou reklamu
* Plugin vás nebude obtěžovat žádným nastavením

**Tento plugin není oficiálním pluginem společnosti WEDOS a.s.!**
